# ООО «МирИгрушек» — ИС магазина игрушек (ДЭ 09.02.07-2-2026, вариант В4)

Полный пакет для развёртывания на чистом **Ubuntu**-сервере (LAMP) одной командой.

## Установка на сервер одной командой

На чистом Ubuntu-сервере выполните:

```bash
curl -fsSL https://raw.githubusercontent.com/skydisgusting/mir-igrushek/main/install.sh | sudo bash
```

Скрипт сам поставит git, склонирует репозиторий в `/opt/mir-igrushek` и запустит установку
(Apache + MySQL + PHP, создание БД, развёртывание сайта).

## Развёртывание вручную (если репозиторий уже скачан)

Скопируйте папку на сервер и выполните **одну команду** из неё:

```bash
sudo bash setup.sh
```

Скрипт:
1. ставит Apache, MySQL, PHP;
2. создаёт БД `mir_igrushek_bd` со всеми таблицами и связями (`init.sql`);
3. делает пользователя **root доступным отовсюду** с паролем `skzskz`, `bind-address = 0.0.0.0`;
4. разворачивает сайт и **убирает приветственную страницу Apache**.

## Доступы

| Что | Значение |
|---|---|
| Сайт | `http://<IP-сервера>/` |
| MySQL Workbench | host = `<IP>`, port = `3306`, user = `root`, pass = `skzskz` |
| База данных | `mir_igrushek_bd` |
| Вход в приложение | логин = e-mail из задания, пароль = `skzskz` (у всех) |

Логин = фамилия латиницей + `@mail.ru`. Примеры:
- Администратор — `vorsin@mail.ru`
- Менеджер — `mikhaylyuk@mail.ru`
- Авториз. клиент — `stepanov@mail.ru`

## Роли
- **Гость** — только просмотр каталога.
- **Авторизированный клиент** — просмотр каталога.
- **Менеджер** — каталог с фильтром/сортировкой/поиском + просмотр заказов.
- **Администратор** — всё + добавление/редактирование/удаление товаров и заказов.

## Структура БД

Справочники: `roles`, `categories`, `suppliers`, `brands`, `units`, `statuses`, `pickup_pts`.
Основные: `accounts` (→roles), `goods` (→units/suppliers/brands/categories),
`orders` (→pickup_pts/accounts/statuses), `order_lines` (→orders/goods).

## Цветовая схема
- Фон `#F5DEB3` (боковая панель), акцент `#DEB887`, фон страницы `#efe7d8`
- Скидка > 17% → строка каталога `#FFDEAD`
- Нет на складе → строка каталога `#D6ECFF`
- Перечёркнутая старая цена: `#e0322f`
