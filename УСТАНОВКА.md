# Инструкция по установке сервера «МирИгрушек»

Пошаговое руководство: что делать и какие данные вводить при настройке сервера
с нуля и развёртывании информационной системы магазина.

---

## 0. Что понадобится

- Виртуальная машина или сервер с **Ubuntu Server 22.04 / 24.04**.
- Доступ в интернет на сервере (для загрузки пакетов).
- На вашем компьютере: браузер и **MySQL Workbench**.

---

## 1. Установка Ubuntu (что вводить в мастере установки)

| Экран мастера | Что выбрать / ввести |
|---|---|
| **Language** | English (или Русский) |
| **Type of install** | Ubuntu Server |
| **Network** | оставить автоматически (DHCP). Запишите **IP-адрес** |
| **Storage / диск** | Use an entire disk → Done → Continue |
| **Profile setup** | Your name / server's name / username / **пароль Ubuntu** |
| **SSH Setup** | поставьте галочку **Install OpenSSH server** |

Дождитесь окончания → **Reboot Now**. Войдите под созданными логином/паролем.

---

## 2. Узнать IP-адрес сервера

```bash
hostname -I
```

Запишите адрес вида `192.168.x.x`.

---

## 3. Установка системы одной командой

```bash
curl -fsSL https://raw.githubusercontent.com/skydisgusting/mir-igrushek/main/install.sh | sudo bash
```

> Скрипт работает автоматически: ставит Apache + MySQL + PHP, создаёт БД, разворачивает сайт.
> Установка занимает 2–5 минут.

По завершении скрипт напечатает:

```
============================================================
  ГОТОВО.
  Сайт:               http://192.168.x.x/
  MySQL Workbench:    host=192.168.x.x  port=3306  user=root  pass=skzskz
  База данных:        mir_igrushek_bd
  Пользователи сайта: логин = email из задания, пароль = skzskz
============================================================
```

---

## 4. Открыть сайт

В браузере откройте `http://<IP-адрес-сервера>/`. Откроется страница входа.

---

## 5. Подключение в MySQL Workbench

1. MySQL Workbench → **➕** рядом с «MySQL Connections».
2. Параметры:

   | Поле | Значение |
   |---|---|
   | **Hostname** | IP-адрес сервера |
   | **Port** | `3306` |
   | **Username** | `root` |
   | **Password** | `skzskz` |

3. **Test Connection** → «Successfully made the MySQL connection».
4. В **SCHEMAS** будет база **`mir_igrushek_bd`**.

## 7. Если что-то не работает

| Проблема | Решение |
|---|---|
| `curl: command not found` | `sudo apt update && sudo apt install -y curl`, затем повторить команду из шага 3 |
| Сайт не открывается | проверьте IP (`hostname -I`); убедитесь, что компьютер и сервер в одной сети |
| Workbench не подключается | `sudo systemctl status mysql`; проверьте порт `3306` и пароль `skzskz` |
| Нужно переустановить | запустите команду из шага 3 ещё раз — она пересоздаёт базу и сайт |
